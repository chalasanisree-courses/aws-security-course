Key insight for the SCS-C03 exam: AdministratorAccess is not the same as root. It is the highest IAM permission level, but IAM itself is a service that runs inside the account — and root sits above that service. Root created IAM. Root can delete all IAM users including the admin. The admin cannot touch root.
