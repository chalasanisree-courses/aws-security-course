Services AWS Services Reference All 12 weeks — revealed weekly ▶

New services revealed each week. Greyed-out rows are coming in future weeks. Filter by scope below.

Scope:
Global Applies account-wide, no region selector
Regional One instance per region
AZ-specific Bound to one availability zone
root gate Root must activate IAM access
free tier Free tier available
paid Costs begin after trial
Service Scope Purpose Used for in this course
IAM
Identity & Access Management
Week 1		 Global Controls who can do what in your AWS account. Manages users, roles, groups, and policies. Applies across all regions. Creating the IAM admin user. Week 8: implementing all 6 Photo Viewer roles.
Global — configured once, affects all regions.
Billing
Billing and Cost Management
root gate
Week 1		 Global Aggregates all AWS usage charges across every region into one monthly bill. CloudWatch billing metrics published to us-east-1 only. $10 CloudWatch billing alarm + $10 AWS Budget.
IAM users cannot access billing until root activates it in Account settings.
CloudTrail
AWS CloudTrail
free tier
Week 1		 Regional Records every API call in your account — who did what, when, from where. Your account's audit log. photoviewer-trail configured all-regions from day one. Week 9: analyzing 8 weeks of real logs.
Regional service configured to collect globally into one S3 bucket.
CloudWatch
Amazon CloudWatch
free tier
Week 1		 Regional Monitors AWS resources. Collects metrics, creates alarms, and triggers automated responses. Billing alarm in us-east-1 (Week 1). Lambda error alarms, structured log analysis (Week 6+).
Billing metrics only exist in us-east-1 — must switch regions to create billing alarm.
S3
Simple Storage Service
free tier
Week 2		 Regional Object storage for files of any type. AWS automatically stores data across multiple AZs — no AZ selection needed. Static Photo Viewer hosting. Photo storage in /assets and /uploads. CloudTrail log bucket.
CloudFront
Amazon CloudFront
free tier
Week 3		 Global Content delivery network with 400+ edge locations. Caches content close to users globally. Serving Photo Viewer content. Privatizing S3 via Origin Access Control (OAC).
EC2
Elastic Compute Cloud
free tier
Week 3		 AZ-specific Virtual servers. Each instance runs in exactly one AZ determined by its subnet. You manage the OS. Hosting the Photo Viewer frontend web server. Inspector scan target.
VPC
Virtual Private Cloud
free tier
Week 3		 Regional Software-defined private network. VPC spans the whole region. Subnets within a VPC are each pinned to one AZ. Network boundary for EC2. Public subnet for web server.
Route 53
Amazon Route 53
paid
Week 3		 Global DNS service. Routes internet traffic to your application. Health checks enable automatic failover. Custom domain for Photo Viewer pointing to CloudFront.
Systems Manager
AWS Systems Manager
free tier
Week 3		 Regional Manages EC2 instances without SSH. Session Manager, patch baselines, run commands remotely. Replacing SSH on EC2. Automated OS patching. Inspector integration.
Lambda
AWS Lambda
free tier
Week 4		 Regional Run code without managing servers. AWS runs functions across AZs automatically. PaaS — you manage code only. Backend API: image listing, uploads, session validation, Rekognition integration.
API Gateway
Amazon API Gateway
free tier
Week 4		 Regional Creates and secures REST APIs. Handles throttling, JWT authentication, and usage plans. REST endpoints for Photo Viewer. JWT Authorizer for token validation. Rate limiting.
Inspector
Amazon Inspector
free tier
Week 4		 Regional Automated CVE scanning for EC2 instances. References NIST NVD. Requires SSM Agent. Scanning EC2 web server for CVEs. Verifying patch compliance after SSM patching.
WAF
AWS Web Application Firewall
paid
Week 5		 Regional Filters web traffic. Blocks SQLi, XSS, and rate-based attacks. Must be us-east-1 for CloudFront. Rate-limiting /login. SQLi and XSS rules on CloudFront (Week 12).
DynamoDB
Amazon DynamoDB
free tier
Week 6		 Regional Fully managed NoSQL database. AWS distributes across AZs automatically. TTL expires old records. Server-side sessions with idle timeout. User accounts table with tier field.
Cognito
Amazon Cognito
free tier
Week 7		 Regional User authentication. Supports Google, Apple, Facebook via Authorization Code + PKCE flow. End-user login. Free vs premium tier. Social login via Google, Apple, Facebook.
IAM Identity Center
AWS SSO
free tier
Week 8		 Global Single sign-on for employee access. Centralizes identity for multiple roles and accounts. Employee login for 4 internal roles: infra admin, web dev, content employee, auditor.
GuardDuty
Amazon GuardDuty
paid
Week 9		 Regional Threat detection. Analyzes CloudTrail, VPC Flow Logs, and DNS logs for malicious activity. Detecting credential exfiltration and suspicious API calls. Sample findings for lab.
Security Hub
AWS Security Hub
paid
Week 9		 Regional Unified security dashboard. Aggregates findings from GuardDuty, Inspector, Config, and Macie. Unified findings view. Posture score comparison — Week 1 vs Week 12.
Secrets Manager
AWS Secrets Manager
paid
Week 9		 Regional Stores and rotates credentials and API keys. Replaces hardcoded values in application code. Replacing hardcoded Lambda env vars with runtime secret retrieval.
AWS Config
AWS Config
paid
Week 10		 Regional Continuously monitors resource configurations. Alerts when they drift from your security baseline. Rules: S3 not public, EC2 managed by SSM, CloudTrail enabled.
Trusted Advisor
AWS Trusted Advisor
free tier
Week 10		 Regional Inspects your account for security, cost, and performance improvements. Full account security review — flagging open ports, public buckets, IAM issues.
KMS
Key Management Service
paid
Week 11		 Regional Creates and manages encryption keys for S3, DynamoDB, EBS, and other services. Encrypting S3 uploads bucket and DynamoDB users table with customer-managed keys.
Macie
Amazon Macie
paid
Week 11		 Regional Machine learning discovery of PII and sensitive data in S3 buckets. Scanning uploads bucket for PII in photo metadata and tags.
Rekognition
Amazon Rekognition
paid
Week 11		 Regional AI image analysis. Detects unsafe content in photos. Runs in a separate account governed by SCP. Auto-scanning uploaded photos. Auto-deleting unsafe images.
AWS Organizations
AWS Organizations
free tier
Week 11		 Global Manages multiple AWS accounts centrally. Service Control Policies restrict what accounts can do. Creating second account for Rekognition. SCP denying all except Rekognition.
