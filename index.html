Week 1 Course foundations + AWS account baseline ▶

Account setup: MFA on root, IAM admin user, CloudTrail all-regions trail, CloudWatch billing alarm, Cost Anomaly Detection. Foundational concepts are covered in the Background section above.

◈ Introduction to AWS — Cloud Models, Services, Shared Responsibility, Well-Architected ◈ AWS Global Infrastructure — interactive diagram ◈ AWS Account Authority Model — interactive diagram ▸ Videos & Labs — see Canvas LMS
Week 2 Static S3 Hosting + Bucket Security ▶

Object storage vs block vs file storage. S3 buckets, three-layer access control, encryption at rest and in transit. Deploy Photo Viewer v1 as a static S3 website. Demonstrate public bucket vulnerabilities: enumeration, confidential file exposure, HTTP only. CloudTrail evidence of anonymous access.

◈ S3 Object Storage and Security — concept notes ▸ Videos & Labs — see Canvas LMS
Week 3 CloudFront CDN + HTTPS Enforcement ▶

Replace direct S3 access with CloudFront distribution. Enforce HTTPS, block direct S3 URLs, implement Origin Access Control. Demonstrate MitM on plain HTTP and compare before/after CloudTrail logs.

◈ CloudFront and Edge Security — concept notes ▸ Videos & Labs — see Canvas LMS
Phase 2
Week 4 EC2 Backend in VPC + DynamoDB ▶

3-tier application architecture — presentation, application, data tiers. VPC design: public and private subnets, security groups, route tables. ALB in public subnet, EC2 application server in private subnet, DynamoDB as data tier. VPC endpoints for DynamoDB, S3, and SSM. SSM Session Manager — no SSH, no port 22. NAT Gateway for bootstrap then deleted. Deliberate vulnerabilities: ALB internet-facing, no rate limiting, overprivileged role supply chain attack demo.

◈ VPC, EC2, and Application Tier Security — concept notes ▸ Videos & Labs — see Canvas LMS
Week 5 Serverless Migration — Lambda + API Gateway ▶

EC2 and ALB replaced with Lambda and API Gateway. Same DynamoDB table, same CloudFront distribution. Eliminates the EC2 attack surface — no OS patching, no SSH, no server management.

▸ Videos & Labs — see Canvas LMS

Service Scope Purpose
IAM
· Identity & Access Management
Week 1		 Global Controls who can do what in your AWS account. Manages users, roles, groups, and policies. Applies across all regions.
Billing
· Billing and Cost Management
root gate Week 1		 Global Aggregates all AWS usage charges across every region into one monthly bill. CloudWatch billing metrics published to us-east-1 only.
CloudTrail
· AWS CloudTrail
free tier Week 1		 Regional Records every API call in your account — who did what, when, from where. Your account's audit log.
CloudWatch
· Amazon CloudWatch
free tier Week 1		 Regional Monitors AWS resources. Collects metrics, creates alarms, and triggers automated responses.
S3
· Simple Storage Service
free tier Week 2		 Regional Object storage for files of any type. AWS automatically stores data across multiple AZs — no AZ selection needed.
CloudFront
· Amazon CloudFront
free tier Week 3		 Global Content delivery network with 400+ edge locations. Caches content close to users globally.
EC2
· Elastic Compute Cloud
free tier Week 4		 AZ-specific Virtual servers. Each instance runs in exactly one AZ determined by its subnet. You manage the OS.
VPC
· Virtual Private Cloud
free tier Week 4		 Regional Software-defined private network. VPC spans the whole region. Subnets within a VPC are each pinned to one AZ.
Systems Manager
· AWS Systems Manager
free tier Week 4		 Regional Manages EC2 instances without SSH. Session Manager, patch baselines, run commands remotely.
ALB
· Application Load Balancer
free tier Week 4		 Regional Layer 7 load balancer. Distributes HTTP/HTTPS traffic across targets. Supports path-based routing and host-based routing.
Internet Gateway
· VPC Internet Gateway
free tier Week 4		 Regional Allows communication between a VPC and the internet. Attached to a VPC — enables public subnets to send and receive internet traffic.
NAT Gateway
· Network Address Translation Gateway
paid Week 4		 AZ-specific Allows private subnet resources to initiate outbound internet traffic without being reachable from the internet. Used for bootstrapping EC2 instances then deleted.
DynamoDB
· Amazon DynamoDB
free tier Week 4		 Regional Fully managed NoSQL database. AWS distributes across AZs automatically. TTL expires old records.
Route 53
· Amazon Route 53
paid		 Global DNS service. Routes internet traffic to your application. Health checks enable automatic failover.
Lambda
· AWS Lambda
free tier		 Regional Run code without managing servers. AWS runs functions across AZs automatically. PaaS — you manage code only.
API Gateway
· Amazon API Gateway
free tier		 Regional Creates and secures REST APIs. Handles throttling, JWT authentication, and usage plans.
Inspector
· Amazon Inspector
free tier		 Regional Automated CVE scanning for EC2 instances. References NIST NVD. Requires SSM Agent.
WAF
· AWS Web Application Firewall
paid		 Regional Filters web traffic. Blocks SQLi, XSS, and rate-based attacks. Must be us-east-1 for CloudFront.
Cognito
· Amazon Cognito
free tier		 Regional User authentication. Supports Google, Apple, Facebook via Authorization Code + PKCE flow.
IAM Identity Center
· AWS SSO
free tier		 Global Single sign-on for employee access. Centralizes identity for multiple roles and accounts.
GuardDuty
· Amazon GuardDuty
paid		 Regional Threat detection. Analyzes CloudTrail, VPC Flow Logs, and DNS logs for malicious activity.
Security Hub
· AWS Security Hub
paid		 Regional Unified security dashboard. Aggregates findings from GuardDuty, Inspector, Config, and Macie.
Secrets Manager
· AWS Secrets Manager
paid		 Regional Stores and rotates credentials and API keys. Replaces hardcoded values in application code.
AWS Config
· AWS Config
paid		 Regional Continuously monitors resource configurations. Alerts when they drift from your security baseline.
Trusted Advisor
· AWS Trusted Advisor
free tier		 Regional Inspects your account for security, cost, and performance improvements.
KMS
· Key Management Service
paid		 Regional Creates and manages encryption keys for S3, DynamoDB, EBS, and other services.
Macie
· Amazon Macie
paid		 Regional Machine learning discovery of PII and sensitive data in S3 buckets.
Rekognition
· Amazon Rekognition
paid		 Regional AI image analysis. Detects unsafe content in photos. Runs in a separate account governed by SCP.
AWS Organizations
· AWS Organizations
free tier		 Global Manages multiple AWS accounts centrally. Service Control Policies restrict what accounts can do.
